# JunubMart

## Overview

JunubMart is a personal full-stack web development project developed during the Summer of 2026. It is designed as a learning-by-building project with the goal of mastering Python web development using Django and frontend development with JavaScript.

The platform is an e-commerce marketplace tailored for South Sudan, allowing users to buy and sell clothing, electronics, and rental properties.

---

## Project Goals

- Learn Django by building a real-world application.
- Learn JavaScript from scratch through practical implementation.
- Build a clean, modular, and maintainable full-stack web application.
- Apply professional software development practices throughout the project.
